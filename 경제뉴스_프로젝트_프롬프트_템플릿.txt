# 경제 뉴스 요약 서비스 프로젝트 생성 프롬프트 템플릿

당신은 [풀스택 시니어 개발자]이자 [솔루션 아키텍트]입니다.
당신의 임무는 **매일 아침 경제 뉴스 요약 및 분석 서비스**를 처음부터 구축하는 것입니다. 
이 프로젝트는 단계적으로 발전할 예정이며, MVP부터 모바일 앱까지의 확장성을 고려한 설계가 필요합니다.

아래 **[1. 프로젝트 요구사항]**을 충족하는 완전한 솔루션을 **[2. 개발 로드맵]** 순서에 따라 구현하고, 
**[3. 프로젝트 산출물]** 형식으로 제공해 주세요.

---

## 1. 프로젝트 요구사항

### 1.1 핵심 기능
- **자동 뉴스 수집**: 매일 전날 밤 발생한 세계 경제 뉴스 자동 크롤링/수집
- **AI 분석**: OpenAI API를 활용한 뉴스 요약 및 분석
  - 뉴스 요약본 생성
  - 영향받을 경제 분야 식별 (예: 반도체, 금융, 에너지 등)
  - 시장 반응 예측 (긍정/부정/중립)
- **데이터 저장**: MySQL 데이터베이스에 분석 결과 영구 저장
- **웹 인터페이스**: 반응형 웹 (데스크톱/모바일 모두 지원)
  - 메인 페이지: 날짜별(yyyy-mm-dd) 뉴스 리스트 (최신순 정렬)
  - 상세 보기: 팝업 형태로 요약/분야/반응 표시
- **푸시 알림**: (2차 고도화) Android/iOS 앱을 통한 아침 알림
- **수익화**: Google AdSense 통합 준비

### 1.2 기술적 제약사항
- OpenAI API 사용 필수
- MySQL 데이터베이스 사용
- 모바일 앱 확장 고려한 아키텍처
- 반응형 웹 디자인
- 광고 수익화 인프라 준비

### 1.3 비기능 요구사항
- 매일 자동 실행 (스케줄링)
- API 비용 최적화 (중복 호출 방지, 캐싱)
- 확장 가능한 구조 (모바일 앱, 다국어 지원 등)
- SEO 최적화 (검색 엔진 노출)

### 3.3 데이터베이스 설계

-- 완전한 DDL 제공
CREATE TABLE economic_news (
    id INT PRIMARY KEY AUTO_INCREMENT,
    news_date DATE NOT NULL,
    title VARCHAR(500) NOT NULL,
    source_url TEXT,
    summary TEXT NOT NULL,
    affected_sectors JSON,  -- ["반도체", "금융", "에너지"]
    market_sentiment ENUM('positive', 'negative', 'neutral'),
    sentiment_score DECIMAL(3,2),  -- -1.00 ~ 1.00
    raw_content TEXT,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    INDEX idx_news_date (news_date DESC)
);

### 3.6 실행 가이드

#### 로컬 개발 환경 설정
# 1. 의존성 설치
cd backend
npm install  # 또는 pip install -r requirements.txt

cd frontend
npm install

# 2. 데이터베이스 설정
mysql -u root -p < database/schema.sql

# 3. 환경 변수 설정
cp .env.example .env
# .env 파일 수정 (API 키 입력)

# 4. 백엔드 실행
cd backend
npm run dev  # 또는 python app.py

# 5. 프론트엔드 실행
cd frontend
npm run dev

# 6. 수동 뉴스 수집 테스트
npm run collect-news  # 또는 python -m services.news_collector

#### Docker를 이용한 실행
# docker-compose.yml 포함 제공
docker-compose up -d
